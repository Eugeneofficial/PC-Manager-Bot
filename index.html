Начало работы с PC Manager Bot

1. Установка бота

Скачайте последнюю версию бота с нашего сайта и распакуйте архив в удобное место.

1. Скачайте PC_Manager_Bot.zip
2. Распакуйте архив
3. Запустите setup.exe

2. Настройка Telegram

Получите токен бота у @BotFather в Telegram:

  1. Найдите @BotFather в Telegram
  2. Отправьте команду /newbot
  3. Следуйте инструкциям для создания бота
  4. Сохраните полученный токен

3. Первый запуск

При первом запуске укажите токен бота и выберите базовые настройки безопасности.

Настройка бота

Конфигурационный файл

Все настройки хранятся в файле config.json:

{
  "token": "ваш_токен_бота",
  "admin_id": "ваш_telegram_id",
  "allowed_users": [],
  "notifications": true,
  "screenshot_quality": "high"
}

Безопасность

Уровни доступа

  • Администратор - полный доступ ко всем функциям
  • Пользователь - базовые команды и мониторинг
  • Гость - только просмотр статуса

Никогда не передавайте токен бота посторонним лицам!
