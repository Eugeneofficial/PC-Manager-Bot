Getting Started with PC Manager Bot

1. Installation

Download the latest version of the bot from our website and extract the archive to a convenient location.

1. Download PC_Manager_Bot.zip
2. Extract the archive
3. Run setup.exe

2. Telegram Configuration

Get the bot token from @BotFather in Telegram:

  1. Find @BotFather in Telegram
  2. Send the command /newbot
  3. Follow the instructions to create the bot
  4. Save the obtained token

3. First Run

When you first run it, specify the bot token and select the basic security settings.

Bot Settings

Configuration File

All settings are stored in the config.json file:

{
  "token": "your_bot_token",
  "admin_id": "your_telegram_id",
  "allowed_users": [],
  "notifications": true,
  "screenshot_quality": "high"
}

Security

Access Levels

  • Administrator - full access to all functions
  • User - basic commands and monitoring
  • Guest - only view status

Never share your bot token with anyone!
